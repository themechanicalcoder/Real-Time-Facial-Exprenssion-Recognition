# Real-Time-Facial-Exprenssion-Recognition

## It is a deep learning model that takes live video from the webcam and predicts the expression of the people in front of it. </br>

## Requirements </br>
1)Tensorflow</br>
2)Python3</br>
3)Keras</br>
